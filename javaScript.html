DOM

// Acessar um elemento pelo seu ID
let elemento = document.getElementById("idElemento");

// Acessar um elemento pelo seu nome
let elementos = document.getElementsByName("nomeElemento");

// Acessar todos os elementos de um tipo específico
let todosElementos = document.getElementsByTagName("p");

// Criar um novo elemento
let novoElemento = document.createElement("p");

// Adicionar um elemento filho a outro elemento
elemento.appendChild(novoElemento);

// Remover um elemento filho de outro elemento
elemento.removeChild(novoElemento);

// Adicionar ou modificar um atributo de um elemento
elemento.setAttribute("atributo", "valor");

// Acessar o valor de um atributo de um elemento
let valorAtributo = elemento.getAttribute("atributo"); // Adicionar um manipulador de eventos a um elemento
elemento.addEventListener("evento", function() { // Código a ser executado quando o evento ocorrer
});

No exemplo acima, foram apresentados alguns trechos de código em JavaScript para manipular o DOM (Document Object Model). Esses trechos podem ser utilizados para interagir com os elementos HTML de uma página web, modificando-os ou adicionando novos conteúdos.

Uso de Eventos

document.getElementById('meuBotao').addEventListener('click', function() { alert('Botão foi clicado!'); });

Sempre que o botão com id 'meuBotao' for clicado, a função anônima será executada, exibindo um alerta com a mensagem "Botão foi clicado!".

Logica de Programação

// Variáveis e Tipos de Dados
var nome = "João";
var idade = 25;
var possuiGraduacao = true;

// Funções
function saudacao(nome) {
return "Olá, " + nome + "!";
}

console.log(saudacao(nome));

// Estruturas de Controle
for (var i = 0; i < 5; i++) { console.log("Executando loop: " + i); }

// Objetos e JSON
var pessoa = {
nome: " Maria", idade: 30, possuiGraduacao: false }; console.log(JSON.stringify(pessoa));

Neste exemplo, variáveis são declaradas para armazenar informações básicas. A função "saudacao" é definida para retornar uma saudação personalizada. Um loop "for" é usado para executar um bloco de código várias vezes.

Um objeto "pessoa" é criado para armazenar informações sobre uma pessoa. Por fim, a função "JSON.stringify()" é usada para converter o objeto em uma string JSON.
