React

O React é uma biblioteca popular de desenvolvimento front-end que se destaca pela sua eficiência e reutilização de código. Sua importância reside em oferecer uma abordagem declarativa e orientada a componentes para a construção de interfaces de usuário interativas e escaláveis.

Através do React, os desenvolvedores podem criar hierarquias de componentes reutilizáveis que facilitam a manutenção e atualização do código. Além disso, o React utiliza um Virtual DOM para atualizar apenas o necessário, resultando em um desempenho superior.

O React também possui uma comunidade ativa e uma vasta variedade de bibliotecas e ferramentas adicionais que ajudam a estender e melhorar suas funcionalidades. O uso do React pode simplificar o desenvolvimento e oferecer uma melhor experiência para os usuários finais.

Em resumo, a importância do React está em sua abordagem orientada a componentes, eficiência no processamento do DOM, reutilização de código e suporte da comunidade, resultando em um desenvolvimento mais eficiente e interfaces de usuário de alta qualidade.

Hooks

import React, { useState } from 'react';

function App() {
const [count, setCount] = useState(0);

return (
< div> < p>Você clicou {count} vezes
< button onClick={()=> setCount(count + 1)}>
Clique aqui


);
}
export default App;

Neste exemplo, usamos o hook useState para criar uma variável de estado chamada count. Quando o botão é clicado, a função setCount é chamada, atualizando o valor de count.

UseState

import React, { useState } from 'react';

function Counter() {
const [count, setCount] = useState(0);

return (
< div>
< p>Você clicou {count} vezes
< button onClick={()=> setCount(count + 1)}>
Clique aqui


);
}

function App() {
return (
< div>
< Counter />
< Counter />

);
}

export default App;

Neste exemplo, criamos um componente chamado Counter que gerencia seu próprio estado usando o hook useState. O componente App renderiza dois componentes Counter, cada um com seu próprio estado.

Projeto

import React, { useState, useEffect } from 'react';
import './App.css';

function App() {
const [todos, setTodos] = useState([]);
const [input, setInput] = useState('');

useEffect(() => {
fetch('https://jsonplaceholder.typicode.com/todos')
.then((response) => response.json())
.then((data) => setTodos(data)); }, []);

const addTodo = (e) => {
e.preventDefault();
setTodos([...todos, { title: input, completed: false }]);
setInput(''); };

const removeTodo = (id) => {
setTodos(todos.filter((todo) => todo.id !== id)); };

const completeTodo = (id) => {
setTodos(
todos.map((todo) =>
todo.id === id ? { ...todo, completed: !todo.completed } : todo ) ); };

return (
< div className="App">
< h1>Tarefas
< form onSubmit={addTodo}>
< input type="text" value={input} onChange={(e)=> setInput(e.target.value)} />
< button type="submit">Adicionar

< ul>
{todos.map((todo) => (
< li key={todo.id}>
< input type="checkbox" checked={todo.completed} onChange={()=> completeTodo(todo.id)} />
< span>{todo.title}
< button onClick={()=> removeTodo(todo.id)}>Excluir ))} ); }

export default App;

Neste exemplo, criamos um aplicativo de gerenciamento de tarefas usando a API JSONPlaceholder. Utilizamos o hook useState para armazenar a lista de tarefas e o valor do campo de entrada. Utilizamos o hook useEffect para buscar a lista de tarefas da API quando o componente é montado.

Ao adicionar uma tarefa, atualizamos a lista de tarefas com a nova tarefa. Ao excluir ou completar uma tarefa, filtramos a lista de tarefas e atualizamos o estado.

Este exemplo demonstra compreensão das melhores práticas do React, incluindo a utilização de hooks, como useState e useEffect, e a criação de componentes funcionais que gerenciam seu próprio estado.

O estilo do aplicativo foi adicionado a um arquivo separado App.css para melhor organização e legibilidade do código. O arquivo App.css pode ser encontrado no mesmo diretório que o arquivo App.js.
